# Home Website

A fully designed personal website showcasing projects, skills, social profiles, and general information. Built to be lightweight, fast, and responsive, and deployed using GitHub Pages.

## Overview

This website serves as a personal homepage with a simple structure and clean layout. It includes navigation, social links, a footer section, and responsive styling. 
## Features

* Multi-page static website
* Responsive layout for mobile and desktop
* Social media integration (Instagram, YouTube, Facebook, GitHub, Twitter)
* Clean visual design with custom styling
* Reusable footer and component-based structure
* Hosted publicly via GitHub Pages

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Bootstrap 
* GitHub Pages for hosting
* Git for regular updates

## How to Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/anshulvishwakarma9/home_website.git
   ```
2. Open the project folder:

   ```bash
   cd home_website
   ```
3. Open any `.html` file in your browser.

No build tools or dependencies are required.

## Deployment

This site is deployed using GitHub Pages:

1. Push all files to a public GitHub repository.
2. Selected branch: `main` (root).

The site will be available at:
`https://anshulvishwakarma9.github.io/home_website/`

## Maintenance

* Update content directly in the HTML files.
* Use `footer.html` or similar partials to avoid repeating code.
* Keep commits small and structured for clean version history.
